# 🎯 Number Guessing Game (Python)

## 📌 Project Description

This is a simple command-line based Number Guessing Game developed using Python.
The program randomly selects a number within a given range, and the user must guess the correct number. The game provides hints such as "Too High" or "Too Low" until the correct number is guessed.

This project was built to practice core Python programming concepts such as loops, conditional statements, input validation, counters, and the use of the random module.

---

## 🚀 Features

* Random number generation within a range
* User input validation using `isdigit()`
* Hint system (Too High / Too Low)
* Range checking
* Guess counter
* Game stops when the correct number is guessed

---

## 🛠️ Technologies Used

* Python 3
* Random Module
